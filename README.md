# ML Resume Screening Project

## Overview
This project aims to automate the process of screening resumes using machine learning techniques. By leveraging various algorithms, we can evaluate resumes based on their relevance to specific job descriptions, ultimately streamlining the hiring process.

## Features
- **Resume Parsing**: Extracting essential information from resumes.
- **Skill Matching**: Identifying key skills that match job requirements.
- **Recommendation Engine**: Rank candidates based on their suitability for the position.

## Getting Started
1. Clone the repository: `git clone https://github.com/Aziz251984/1.git`
2. Install necessary dependencies: `pip install -r requirements.txt`
3. Run the script to start the screening process: `python main.py`
